mac下tomcat安装与配置

背景

首先说说为什么要在mac上安装tomcat。最近，在上一门软件测试技术的课，课上需要安装一个jfroum论坛进项测试，在本机上就需要安装一个服务器，所以使用的是tomcat。实验课上很快就安装完成，之后发现Windows平台的安装很简单，手里有一台mac，就想在自己的电脑上装一个，以后上实验课就带自己的电脑，毕竟实验室的电脑超级卡，浪费时间坐在实验室，还不如快点做完回去学习。遂开始安装，但是在安装的过程中并不是那么简单的,下面就介绍一下安装的过程。

1.处理资源包

首先需要已经安装JDK！！！这个到apache官方主页下载mac版本的文件包，可以选tar.gz格式的。下载完成之后解压，重命名为Tomcat并移动到/Library下面。如果在Finder中没有找到的话，可以按住shift+command+G这个组合键，在框里面输入/然后进入根目录，在里面找到资源库。注意的是，进行拷贝操作需要root权限，所以在此需要输入密码。

2.修改文件权限

打开电脑上自带的终端程序，在里面输入 

sudo chmod 755 /Library/Tomcat/bin/*.sh

将bin目录了下的shell脚本的权限修改，否则会因为权限不够而执行失败。此处也会提示输入密码。

3.启动Tomcat

在上一步执行成功之后，再在终端中输入以下命令 

cd /Library/Tomcat/bin

来切换到刚才复制文件的位置，在输入 

sudo sh startup.sh

若出现以下提示则表示安装并运行成功 

Using CATALINA_BASE:   /Library/Tomcat
Using CATALINA_HOME:   /Library/Tomcat
Using CATALINA_TMPDIR: /Library/Tomcat/temp
Using JRE_HOME:        /Library/Java/JavaVirtualMachines/jdk1.8.0_20.jdk/Contents/Home
Using CLASSPATH:       /Library/Tomcat/bin/bootstrap.jar:/Library/Tomcat/bin/tomcat-juli.jar
Tomcat started.

4.验证服务器是否运行

打开浏览器，输入localhost:8080回车，如果看到Apache Tomcat则表示运行成功！

5.附加

以上运行的一个叫startup.sh的shell脚本是每次开启服务器都需要运行的程序，如果在目录中每次斗殴输入绝对路径在运行的话稍显麻烦，得意与OS X系统的基于UNIX这一特性，加之又学习了Linux这个系统，就想起来把启动，停止和重启服务写成一个脚本。并且在PATH这个环境变量的路径中加入，以后就可以直接输入一个简单的命令来启动或停止服务器了。所编写的脚本如下： 

#!/bin/bash

case $1 in
    "start")
        sudo sh /Library/Tomcat/bin/startup.sh
        echo -e "\nHey!!! Dear mao.Tomcat Started!!!!"
    ;;
    "stop")
        sh /Library/Tomcat/bin/shutdown.sh
        echo -e "\nHey!!! Dear mao.Tomcat Stopped!!!!"
    ;;
    "restart")
        sh /Library/Tomcat/bin/shutdown.sh
        sudo sh /Library/Tomcat/bin/startup.sh
        echo -e "\nHey!!! Dear mao.Tomcat Restarted!!!!"
    ;;
    *)
        echo -e "\nUsage: start|stop|restart"
    ;;
esac

exit 0

将上面的代码复制到文本编辑器，然后保存，这里建议使用Sublime，然后保存名字为tomcat不要大写，也不要后缀。然后再给文件赋予执行权限（首先切换到文件所在的目录）： 

chmod 777 tomcat

然后给这个文件移动到以下目录 

echo $PATH  
/usr/local/bin:/usr/bin:/bin:/usr/sbin:/sbin:/usr/local/git/bin

目录是以：问分割的，推荐移动到/usr/local/bin目录了下，其他目录会需要root权限或者其他原因失败，如果不会在终端中移动文件，那就还是按住上述的组合键，输入目录之后，把文件拖拽进去。至此，就完成了脚本。下面可以试试命令：

1.tomcat start 启动服务
2.tomcat stop 停止服务
3.tomcat restart 重启服务
